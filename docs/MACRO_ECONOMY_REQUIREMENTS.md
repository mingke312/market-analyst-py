# 宏观经济数据收集系统 - 需求文档

## 1. 项目概述

**项目名称：** 宏观经济数据收集系统

**目标：** 收集并整理全球宏观经济数据，为资产配置分析提供数据支撑

**用户：** 何甜（个人投资者，宏观分析师方向）

---

## 3. 数据质量要求

- **完整性：** >= 90%
- **时效性：** 实时或当日
- **准确性：** 与官方数据一致
- **稳定性：** 99% 可用率

---

## 4. 输出格式

### JSON格式

```json
{
  "date": "2026-02-28",
  "type": "macro_economy",
  "data": {
    "gdp": {"value": 126.06, "yoy": 5.0, "quarter": "2024Q4"},
    "cpi": {"value": 0.2, "yoy": 0.2, "month": "2025-01"},
    "central_bank": [
      {"name": "7天逆回购利率", "value": "1.50%"}
    ]
  }
}
```

## 5. 实施优先级

| 优先级 | 数据类别 | 说明 |
|--------|----------|------|
| P0 | GDP、CPI、PPI、央行利率 | 核心指标 |
| P1 | PMI、消费、投资、工业 | 重要经济指标 |
| P2 | 房地产、进出口、货币供应 | 补充数据 |
| P3 | 全球市场、大宗商品、外汇 | 资产配置参考 |
